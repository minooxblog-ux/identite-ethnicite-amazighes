# Amazigh Identity: Digital Construction of Cultural Heritage

An academic article exploring how digital technologies are reshaping Amazigh cultural identity and heritage preservation in the modern era.

## 🚀 Live Site
[https://identiteamazighe.vercel.app](https://identiteamazighe.vercel.app)

## 📋 Overview
This project presents a scholarly analysis of Amazigh identity construction in digital spaces, featuring:

- **Server-Side Rendering (SSR)** with Next.js for optimal SEO
- **Comprehensive SEO** with meta tags, Open Graph, Twitter cards, and structured data
- **Dark/Light Mode** with system preference detection
- **Text-to-Speech** functionality for accessibility
- **Responsive Design** with mobile-optimized table of contents
- **Reading Progress** tracking
- **Interactive Footnotes** with back-to-text navigation

## 🛠️ Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel
- **Content:** Academic article in French

## 🏗️ Architecture
```
app/
├── layout.tsx          # Root layout with SEO metadata
├── page.tsx            # Main article page
├── globals.css         # Global styles
├── robots.ts           # Robots.txt generation
├── sitemap.ts          # Sitemap.xml generation
├── contexts/           # React contexts
├── components/         # Reusable components
├── hooks/              # Custom hooks
└── data/               # Article content data
```

## 🚀 Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## 📊 SEO Features
- Server-side rendering ensures article content is crawlable
- Comprehensive meta tags for search engines
- Open Graph and Twitter Card support
- JSON-LD structured data for rich snippets
- Automatic robots.txt and sitemap.xml generation
- Canonical URLs and proper language attributes

## 🎨 Features
- **Dark Mode:** Automatic system preference detection with manual toggle
- **Accessibility:** Screen reader support, keyboard navigation, high contrast
- **Mobile-First:** Responsive design with collapsible table of contents
- **Performance:** Optimized fonts, lazy loading, and efficient rendering
- **Print Styles:** Clean print layout for academic use

## 📝 Content
The article examines:
- Theoretical frameworks of identity and ethnicity in digital contexts
- Historical context of Amazigh identity in Morocco
- Analysis of digital platforms and their role in identity construction
- Empirical findings from surveys and content analysis
- Implications for cultural preservation and digital heritage
