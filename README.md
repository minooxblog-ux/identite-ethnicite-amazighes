# Amazigh Identity: Digital Construction of Cultural Heritage

An academic exploration of how digital technologies are reshaping Amazigh cultural identity and heritage preservation in the modern era.

[![Live Demo](https://img.shields.io/badge/Live-Demo-d48534.svg?style=for-the-badge)](https://identiteamazighe.vercel.app)

## 📖 Overview

This project is a digital academic publication exploring the intersection of Amazigh (Berber) cultural heritage and modern digital transformation in North Africa. It is designed with a focus on readability, typography, and accessibility.

## ✨ Features

- **Academic Typography**: Elegant serif font stack (Georgia, Times New Roman, Noto Serif) optimized for long-form reading.
- **Interactive Footnotes**: Highlighted and animated footnote references for an enhanced academic reading experience.
- **Reading Progress**: Fixed top-bar progress indicator to track reading journey.
- **Dark Mode Support**: Seamless light and dark mode toggling tailored for eye comfort.
- **SEO & Accessibility**: Fully optimized with Next.js Metadata API and structured JSON-LD Article schema.
- **Custom Theming**: Custom "Amazigh" color palette integrating traditional colors into the Tailwind v4 theme.

## 🚀 Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🛠️ Getting Started

First, clone the repository and install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
